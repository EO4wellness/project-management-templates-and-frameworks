# Initiation Templates

This folder contains tools for starting a project with clarity and alignment.  
These templates support early communication, stakeholder alignment, and cross‑team understanding—bridging gaps before they appear.

Where appropriate, I've included **Templates*, which are fill-in-the-blanks documents, **Checklists** which provide a quick, but time-saving, answer to skim over and answer the "Did I forget anything?" question, and **Thinking Guides** which offer deeper questions to help with reason through the initiation work.  I repeate these in our project close as well, as so we learn from each project, adapting and growing, along the way. 

Contents will include:
- Project Charter:  Template | Checklist | Thinking Guide
- Stakeholder Register: Template | Checklist | Thinking Guide
- Communication Plan
- RACI Matrix
