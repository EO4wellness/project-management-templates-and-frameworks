# Communication Plan Checklist

This checklist ensures the Communication Plan is complete, aligned with stakeholder needs, and ready for use during project initiation and early planning.

---

## 1. Purpose & Objectives
- [ ] Purpose of the communication plan is clearly stated  
- [ ] Communication objectives are defined and aligned with project goals  
- [ ] Plan supports transparency, alignment, and decision-making  
- [ ] Scope boundaries and change‑control messaging are included  

---

## 2. Stakeholder Communication Requirements
- [ ] All key stakeholders are listed  
- [ ] Information needs are documented for each stakeholder/group  
- [ ] Level of detail required is defined  
- [ ] Preferred communication channels are identified  
- [ ] Communication frequency is appropriate and realistic  
- [ ] Communication owners are assigned  

---

## 3. Communication Methods & Channels
- [ ] All communication channels are listed (email, meetings, dashboards, etc.)  
- [ ] Purpose of each channel is clear  
- [ ] Audience for each channel is defined  
- [ ] Frequency is documented  
- [ ] Ownership is assigned  

---

## 4. Meeting Cadence
- [ ] Recurring meetings are listed  
- [ ] Purpose of each meeting is clear  
- [ ] Participants are identified  
- [ ] Frequency and format are defined  
- [ ] Meeting owners are assigned  

---

## 5. Escalation Path
- [ ] Escalation levels are defined  
- [ ] Triggers for each level are documented  
- [ ] Escalation recipients are identified  
- [ ] Expected response times are included  
- [ ] Escalation path aligns with authority structures  

---

## 6. Communication Responsibilities
- [ ] Roles and responsibilities are clearly assigned  
- [ ] PM responsibilities are documented  
- [ ] Sponsor responsibilities are documented  
- [ ] SME or team lead responsibilities are documented  
- [ ] Notes capture any special considerations  

---

## 7. Key Messages
- [ ] Core project messages are defined  
- [ ] Scope boundaries are included  
- [ ] Change‑control expectations are included  
- [ ] Timeline and milestone expectations are included  
- [ ] Messaging is consistent across all channels  

---

## 8. Tools & Documentation
- [ ] Tools for storing and sharing communication artifacts are listed  
- [ ] Access expectations are documented  
- [ ] Version control approach is defined  
- [ ] Meeting notes repository is identified  

---

## 9. Change Control Summary
- [ ] High‑level change‑control process is included  
- [ ] Approval authority is documented  
- [ ] Evaluation criteria are summarized  
- [ ] Communication expectations for change decisions are defined  

---

## 10. Out‑of‑Scope Request Communication Pathway
- [ ] Submission method for out‑of‑scope requests is defined  
- [ ] Logging and tracking process is documented  
- [ ] Evaluation workflow is clear  
- [ ] Approval path is identified  
- [ ] Communication back to requester is defined  
- [ ] Expected response times are included  
- [ ] Pathway aligns with the Project Charter’s governance rules  

---

## 11. Completeness & Quality Check
- [ ] Plan is internally consistent  
- [ ] All fields are filled in or intentionally marked N/A  
- [ ] Information is current and validated with stakeholders  
- [ ] Communication expectations are realistic and sustainable  
- [ ] Version history is updated  
- [ ] Plan is ready for sponsor or PMO review  

---

# Final Review
- [ ] All communication requirements are validated with stakeholders  
- [ ] Escalation paths are clear, realistic, and aligned with authority structures  
- [ ] Out‑of‑scope communication pathway is complete and consistent with the Project Charter  
- [ ] Meeting cadence supports project needs without overloading the team  
- [ ] Communication responsibilities are assigned and understood  
- [ ] Key messages are consistent across all channels  
- [ ] Tools and documentation locations are accessible to all relevant stakeholders  
- [ ] Change‑control summary aligns with the project’s governance rules  
- [ ] Checklist is complete and the Communication Plan is ready for sponsor or PMO review
