# Planning Templates and Tools

This folder contains tools for planning any project with structure, alignment, and predictable execution. These templates support early workflow design, estimation, and team coordination—turning the project vision into actionable plans and shared understanding.

Where appropriate, I've included **Templates**, which are fill‑in‑the‑blanks documents; **Checklists**, which provide a quick way to answer the “Did I forget anything?” question; and **Thinking Guides**, which offer deeper prompts to help reason through planning decisions. These can be adapted for use in any project management framework and are designed to be lightweight, practical, and easy to integrate into real‑project environments. Examples of tools and ecosystems where these apply include Microsoft 365, Google Workspace, ClickUp, Jira, and hybrid setups.
