# Kickoff Meeting Templates and Tools

This folder contains tools for preparing and running a project kickoff meeting with clarity, alignment, and intentional collaboration. The kickoff meeting is the first major activity in the Planning phase and sets the tone for how the team will work together throughout the project.

Where appropriate, I've included **Templates**, which are fill‑in‑the‑blanks documents; **Checklists**, which provide a quick way to answer the “Did I forget anything?” question; and **Thinking Guides**, which offer deeper prompts to help reason through facilitation, alignment, and team dynamics. These tools are designed to be lightweight, practical, and adaptable to any project environment, including Microsoft 365, Google Workspace, ClickUp, Jira, hybrid teams, and distributed teams.

## Purpose

The kickoff meeting is used to:
- align the team on goals, scope, and expectations  
- establish roles and responsibilities  
- introduce collaboration norms and communication pathways  
- surface early questions, risks, and assumptions  
- set the tone for inclusive, accessible participation  
- build shared understanding before execution begins  

These tools help project managers prepare thoroughly, facilitate effectively, and follow up with clarity. They also support PM judgment by providing space for private observations about stakeholder dynamics, communication styles, and early confidence signals.

This folder will continue to evolve as the toolkit grows.
